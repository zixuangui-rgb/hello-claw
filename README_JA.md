## はじめに

本プロジェクトは OpenClaw の包括的なチュートリアルです。この強力なコマンドラインAIアシスタントシステムをゼロからマスターする手助けをします。OpenClaw を素早く使いこなして生産性を向上させたい方も、その仕組みを深く理解して独自のバージョンを構築したい方も、本チュートリアルが明確な学習パスを提供します。

**2つのコアモジュール：**

1. **Clawを迎える（ユーザーガイド）**：OpenClaw のクイックスタート、設定・使用・拡張方法を学ぶ
2. **Clawを構築する（開発者ガイド）**：OpenClaw の内部構造を深く理解し、カスタマイズをマスターする

**対象者：**

- いつでも使えるAIコマンドラインアシスタントが欲しい開発者
- Telegram/Feishu 経由でAIをリモート制御したい効率化志向の方
- OpenClaw のスキルシステムや自動化機能に興味がある学習者
- 自分だけのAI Agentツールを理解・構築したい技術愛好家

**学習パス：**

- 初心者：第1部「Clawを迎える」から始めて、段階的に使い方をマスター
- 開発者：第2部「Clawを構築する」に進んで、実装原理を理解
